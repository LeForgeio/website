The Problem with Workflow Automation

Low-code platforms are great for connecting services, but struggle when you need real computing power:

  • Complex math and statistical analysis
  • AI/LLM integration with local models
  • PDF generation and manipulation
  • OCR and image processing
  • Encryption and secure data handling
  • Vector search and embeddings

LeForge is the Solution

A modular microservices platform that adds compute capabilities to any workflow tool:

  • Pre-built ForgeHooks for common tasks
  • Native nodes/modules for your platform
  • Self-hosted on your infrastructure
  • No vendor lock-in or data leaving your network
  • Horizontally scalable with Docker/K8s
  • Open source and extensible

Up and Running in Minutes

LeForge is designed for simplicity. Deploy once, use everywhere.

1

Deploy with Docker

Single command deployment. docker-compose up and you're running all services with Kong API Gateway.

2

Install Platform Nodes

Add LeForge nodes to n8n, Make modules, or configure webhooks in Power Automate and Zapier.

3

Build Powerful Workflows

Combine AI, data processing, and automation. Your workflows now have compute superpowers.

Simple REST API

Every ForgeHook exposes a clean REST API. Use the SDK, native platform nodes, or call directly via HTTP.

  • OpenAPI/Swagger documentation
  • JavaScript, Python, and .NET SDKs
  • API key authentication
  • Rate limiting and caching built-in
