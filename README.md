# LeForge Website

Marketing website for [LeForge](https://github.com/LeForgeio/leforge) - the self-hosted AI & compute platform for workflow automation.

## Deployment

This site is deployed on [Vercel](https://vercel.com).

### Local Development

Simply open `index.html` in a browser, or use a local server:

```bash
npx serve .
```

### Deploy to Vercel

1. Push to GitHub
2. Connect repository to Vercel
3. Configure custom domain (leforge.io)

## Structure

```text
├── index.html       # Main landing page
├── vercel.json      # Vercel configuration
└── README.md        # This file
```
